# Aqua — минималистичный браузер

Браузер на движке Chromium (Electron) с чистым тёмно-синим интерфейсом.
Весь трафик идёт через ваш собственный сервер: браузер настроен на
прокси, поэтому сайты видят IP сервера, а не ваш домашний.

## Что уже готово

- Вкладки, адресная строка с поиском, кнопки назад/вперёд/обновить.
- Светлая и тёмная темы — подхватываются из настроек macOS.
- **Флот прокси** — несколько серверов выхода, переключение в один клик
  по чипу «Aqua» в адресной строке или горячей клавишей `Cmd/Ctrl+Shift+P`.
- **Хранилище серверов** (шестерёнка справа) — добавить/изменить/удалить
  сервер, вставить список строками `host:port:логин:пароль`, бейджи
  задержки по каждому узлу, тумблер «Прямое соединение».
- **Гео-выхода** — пилюля показывает флаг, город и IP, который видит мир;
  клик копирует IP. Обновляется при каждом переключении сервера.
- **Kill-switch** — если туннель падает, весь трафик замораживается
  (замок краснеет «PAUSED»), настоящий IP не утекает.
- **Аура выхода** — фоновое свечение перекрашивается под страну сервера.

## Несколько серверов (флот)

`config.json` хранит массив `proxies` и `activeId`. Пример — в
`config.example.json`. Добавлять серверы удобнее прямо в приложении
(шестерёнка → «Добавить сервер» или «Вставить списком»), но можно и руками:
каждый сервер — это `{id, label, host, port, user, pass, color, country}`.
Активный сервер задаётся полем `activeId`; `directMode: true` — работать
без прокси.

## Установка на Mac

Нужен установленный **Node.js** (если нет — поставьте с nodejs.org, LTS).

```bash
# 1. скачать код
git clone <адрес-вашего-репозитория> aqua-browser
cd aqua-browser

# 2. установить зависимости (скачает Electron для macOS)
npm install

# 3. создать конфиг из шаблона и вписать адрес сервера + пароль прокси
cp config.example.json config.json
#   откройте config.json и заполните host / pass (значения я пришлю отдельно)

# 4. запустить
npm start
```

## Собрать в приложение (.dmg)

Чтобы получить обычное приложение Aqua.app, которое запускается двойным
кликом и лежит в «Программах»:

```bash
npm run dist
```

Готовый файл появится в папке `dist/`. Первый запуск неподписанного
приложения: правый клик по Aqua → «Открыть» (macOS один раз переспросит).

## Как это устроено

| Файл | Назначение |
|---|---|
| `main.js` | главный процесс: окно, вкладки, подключение прокси |
| `preload.js` | безопасный мост между интерфейсом и системой |
| `ui/index.html` | внешний вид панели (вкладки, адресная строка) |
| `ui/renderer.js` | логика интерфейса |
| `config.json` | адрес сервера и пароль прокси (не попадает в git) |

Прокси-сервер (принимает трафик на стороне сервера) лежит отдельно —
в проекте `aqua-proxy`.
